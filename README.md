This program uses cv2 to give an estimated value of the angle of the sun relative the ground( 0 degrees).
This program first asks the user to set up the camera a perpindicular maginitude away from the ground with the object of choice layed on top. The camera
then measures the dimensions in pixels or the grid of the background. The user is then asked to place the object perpinduclar to the surface to give the angle.
